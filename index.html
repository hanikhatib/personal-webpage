Bringing Photos to Life with Neural Radiance Fields (NeRF)

Neural Radiance Fields (NeRFs) transform 2D images into interactive 3D worlds.

Using NVIDIA's Instant NeRF software developer kit (SDK) and a custom GCP virtual machine setup, I created photorealistic 3D reconstructions of vibrant scenes with out out-of-distribution perspectives. Each scene rendered in hours (a task which previously would have taken days or weeks) using less than 200 images.
